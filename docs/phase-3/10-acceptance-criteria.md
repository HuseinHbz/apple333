# معیارهای پذیرش فاز ۳

- دستگاه سریال‌دار با IMEI تکراری قابل دریافت نیست و تاریخچهٔ lifecycle کامل دارد.
- دریافت از PO تا available بدون ورود دستی خارج از audit قابل انجام است.
- انتقال فقط از statusهای مجاز عبور می‌کند، sender/approver/receiver ثبت می‌شوند و device در انتقال فروخته نمی‌شود.
- رزرو timeout می‌شود و موجودی را آزاد می‌کند؛ available هیچ‌گاه منفی نیست.
- نقش Sales نمی‌تواند دادهٔ شعبهٔ دیگر، IMEI کامل یا تأیید transfer را ببیند/انجام دهد.
- Admin dashboard موجودی، ارزش، low stock و مقایسهٔ شعب را نمایش می‌دهد؛ گزارش‌ها با دادهٔ عملیاتی سازگارند.
- آزمون 100k device و حجم log هدف، IMEI search p95 زیر 300ms را تأیید می‌کند.
- تمام mutationها audit event، request ID و تست خودکار دارند؛ UAT انبار/شعبه امضاشده است.
